## 背景故事

`Discuz! Q`项目由于是从 0 到 1，介于我们的目标，如果从第一行代码开始编写，是极为庞大的工程。想想`Discuz!X`，代码量依赖 10 多年的时间的积累，才完善出各种工具类、自己的框架及插件机制等。
